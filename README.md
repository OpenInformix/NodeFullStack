## The Nodejs REST API 
This is a very basic sample REST API created by using Nodejs. In this we used middleware concept for the routers.


#### start the node API server
```bash
npm start
```

### API basic usage
Then on the browser
```bash
http://localhost:3000/
http://localhost:3000/order
http://localhost:3000/product
http://localhost:3000/order
```
